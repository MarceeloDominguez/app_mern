## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/app_mern.git
```

Go to the project directory client

```bash
  cd app_mern
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
 
Go to the project directory server

```bash
  cd app_mern
  cd server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
